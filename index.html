Curve Tracer - Bachelor's Thesis

This project is part of my Bachelor's Thesis at the University of Seville, where I developed a program in both MATLAB and Python to create a Graphic User Interface (GUI) for characterizing diodes and MOSFET transistors using the National Instruments myDAQ data acquisition system.

The GUI enables users to plot the characteristic curves of diodes and MOSFETs, and includes a curve fitting functionality that calculates fitting coefficients, their errors, and the Pearson correlation coefficient (R²). Additionally, the application allows users to export both the figures and raw data for further analysis.

Key features of the project include:

  • Help Menu: Integrated help system to guide the user through the application's functionalities.
  • Sampling Methods: Support for multiple sampling methods, including Normal and Sweep sampling modes.
  • Device Detection: Automatically detects the myDAQ device and confirms proper connection.
  • Export Data: Allows data and figures to be exported for further analysis and reporting.

The project consists of several components:

  • MATLAB Curve Tracer: Full code for the GUI developed using App Designer in MATLAB.
  • Python Curve Tracer: Python version of the GUI built with tkinter, providing the same functionalities.
  • Scripts: Simplified scripts to obtain diode and MOSFET curves without a GUI for quick measurements.
  • Packed App: A standalone MATLAB app that does not require App Designer to run, simplifying user access.

The full code and documentation for this project can be found on GitHub, and the thesis document is available through the following links:

  • Curve Tracer GitHub Repository
  • Final Thesis Document

Technical Overview:

The project was developed with the goal of providing a comprehensive and easy-to-use application for analyzing electronic components. It integrates the NI myDAQ hardware with advanced data analysis tools in MATLAB and Python, making it ideal for both academic and professional use.

Professional Filter Designer (Python GUI)

This project consists of a Python-based application for the design and analysis of analog and digital filters, featuring an interactive Graphical User Interface (GUI) built with tkinter and advanced signal processing using NumPy and SciPy.

The application allows users to design filters such as Butterworth, Chebyshev Type I & II, Elliptic, and Bessel, supporting multiple configurations including low-pass, high-pass, band-pass, and band-stop filters in both analog and digital domains.

The tool provides a complete visualization and analysis environment, enabling users to evaluate filter performance through multiple representations:

  • Magnitude Response: Logarithmic frequency response in dB.
  • Phase Response: Unwrapped phase analysis.
  • Poles and Zeros: Complex-plane visualization for stability analysis.
  • Impulse and Step Responses: Time-domain characterization.
  • Group Delay: Frequency-dependent delay behavior.
  • Noise Simulation: Evaluation of filter performance under additive white Gaussian noise (AWGN).

Key features of the project include:

  • Automatic Filter Order Estimation: Uses SciPy optimization routines (buttord, cheb1ord, ellipord, etc.) to compute the minimum order that satisfies design specifications.
  • Flexible Frequency Input: Supports both single cutoff frequencies and band definitions.
  • Transfer Function Visualization: Displays the filter in analytical form (H(z) or numerator/denominator representation).
  • Multi-Tab Visualization: Organized interface with dedicated tabs for each analysis domain.
  • Project Persistence: Save and load configurations using JSON files.
  • Integrated Help System: Built-in documentation describing filter types and usage.

The application also includes a signal simulation module, where a sinusoidal signal is corrupted with controlled noise (based on a configurable SNR) and filtered in real time, allowing direct comparison between noisy and filtered signals.

The full source code is available on GitHub:

Technical Overview:

This project demonstrates a complete integration of digital signal processing (DSP) theory with practical software development. It combines filter synthesis, frequency-domain and time-domain analysis, and stochastic signal modeling within a user-friendly interface, making it suitable for both academic learning and engineering prototyping.

Signal Analysis Pro – Python DSP Laboratory Tool

This project is a comprehensive Python-based signal analysis application designed as an academic and engineering tool for studying Digital Signal Processing (DSP). It features an interactive Graphical User Interface (GUI) developed with tkinter, integrating advanced numerical computation via NumPy, SciPy, and Matplotlib.

The application enables users to generate, analyze, and process signals in both time and frequency domains, offering a structured environment suitable for laboratory work, teaching, and self-learning.

Core functionalities include:

  • Signal Generation: Supports sine, cosine, square, and sawtooth waveforms with configurable amplitude, frequency, phase, sampling frequency, and duration.
  • Time & Frequency Analysis: Real-time visualization of signals alongside their spectral representation using FFT and frequency-domain plots.
  • Fourier Methods: Includes Discrete Fourier Transform (DFT) reconstruction and conceptual modules for Fourier series and continuous Fourier transform.
  • Spectrogram Visualization: Time-frequency analysis for non-stationary signals.

The application also provides a complete framework for studying sampling theory:

  • Sampling Analysis: Automatic verification of the Nyquist criterion and aliasing detection.
  • Sampling Visualization: Graphical representation of discrete samples over continuous signals.
  • Sinc Reconstruction: Ideal bandlimited reconstruction using sinc interpolation.

Signal processing capabilities are further extended with filtering and noise analysis:

  • Digital Filter Design: Implementation of configurable Butterworth filters (low-pass, high-pass, band-pass).
  • Zero-phase Filtering: Uses forward-backward filtering (filtfilt) to eliminate phase distortion.
  • Noise Injection & SNR Estimation: Adds Gaussian noise and computes the resulting Signal-to-Noise Ratio (SNR).

Additional features include:

  • Custom Signal Import: Load external datasets from CSV files.
  • PDF Report Generation: Export basic analysis summaries using the ReportLab library.
  • Academic Mode: Placeholder modules for guided exercises and theoretical explanations (in development).
  • Integrated Help System: Built-in documentation interface for user guidance (in development).

The application is structured as a modular DSP laboratory environment, combining signal synthesis, spectral analysis, sampling theory, filtering, and stochastic signal modeling within a unified interface.
